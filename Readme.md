# AcoustiCAD

A professional desktop application for designing audio systems in commercial and residential venues.

## Features

- **Zone-based Design**: Draw and manage audio zones on floor plans
- **Speaker Placement**: Intelligent speaker placement with coverage visualization
- **Scale Calibration**: Accurate measurements using real-world scaling
- **Speaker Profiles**: Comprehensive database of speaker specifications
- **Report Generation**: Professional reports for installation and documentation
- **Project Management**: Save, load, and organize multiple projects

## Installation

### Windows
1. Download the latest `AcoustiCAD-*-Windows-Setup.exe` from [Releases](https://github.com/FreeForgeLabs/AcoustiCAD/releases)
2. Run the installer as administrator
3. Follow the installation wizard
4. Launch from Start Menu or Desktop shortcut

### macOS
1. Download the `.dmg` file
2. Drag the app to Applications folder
3. Launch from Applications or Launchpad

## Quick Start

1. **Create a New Project**: File → New Project
2. **Load Background**: Load a floor plan or architectural drawing
3. **Calibrate Scale**: Set real-world measurements for accuracy
4. **Draw Zones**: Use the drawing tool to define audio zones
5. **Place Speakers**: Switch to Plotter tab and position speakers
6. **Generate Reports**: Export professional documentation

## System Requirements

- **Operating System**: Windows 10+, macOS 10.14+, or Linux
- **RAM**: 4GB minimum, 8GB recommended
- **Storage**: 500MB free space
- **Display**: 1024x768 minimum, 1920x1080 recommended

## File Formats Supported

- **Images**: PNG, JPG, BMP, TIFF
- **Documents**: PDF (background images)
- **Projects**: JSON (native format)
- **Exports**: PNG, JPG, PDF, HTML reports
